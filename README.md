# Rebase

Rebase is a self-hosted analytics workspace for teams that need a private, operationally controlled reporting surface.

## Coolify Deployment

The included Coolify configuration runs the Rebase service with production settings:

- Service name: `Rebase`
- Public URL: `https://rebase.app.t3ai.xyz`
- Application database: PostgreSQL
- Site name: `Rebase`

Use `docker-compose.coolify.yml` for the Coolify service configuration.
